# Heal U: Loading Screen Walkthrough

We have upgraded the loading screen of the application to feature a premium, healthcare-themed mechanical assembly animation.

## Key Enhancements

### 1. Transformer-Style Hospital Construct Animation
- Built with high-fidelity React components using Framer Motion.
- **Stage 1 (15%):** The **Foundation Base** slides up from the bottom with blinking blue-green hydraulic nodes.
- **Stage 2 (40%):** The **Left and Right Building Wings** swing and fold in from the sides with active cyber-circuit lines, while the **Center Pillars** scale up.
- **Stage 3 (65%):** The **Roof Cap** drops down and clicks into place.
- **Stage 4 (85%):** The **Glowing Medical Cross** constructs line-by-line in the center and glows with cyan/indigo neon drop-shadows.

### 2. Ambient Medical Background (No Grid)
- Replaced the hard grid overlays with a smooth, radial dark gradient (#090d16 → #03050a).
- Added two giant, blurred glowing **aurora light blobs** (Cyan & Indigo) that drift and scale slowly in the background.
- Kept the subtle **cyan ECG Heartbeat line** pulsing across the screen to tie directly into the medical concept.

### 3. Elegant Simple Typography
- **App Title:** Styled as `Heal U` in a simple, light system-sans font with the `U` highlighted in semi-bold cyan.
- **Loading Log & Percentage:** Changed the console monospace font to a clean, elegant sans-serif, using soft slate colors (`text-slate-400` and `text-cyan-400/80`).
